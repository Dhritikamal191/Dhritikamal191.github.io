Customer Churn Prediction

End-to-end ML project for predicting customer churn risk.

🚀 Streamlit App 💻 GitHub Repo 📄 Research Paper

Project Overview

Developed a machine learning model to predict customer churn using classification algorithms and SHAP interpretability.

Technologies Used

Python Scikit-Learn XGBoost Streamlit SQL

Key Insights

  • Customers with month-to-month contracts had higher churn.
  • Balance and geography significantly impacted retention.
  • SHAP improved interpretability of model decisions.
