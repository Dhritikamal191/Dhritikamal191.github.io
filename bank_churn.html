Bank Customer Churn Analysis

End-to-end Data Analytics project for customer churn.

🚀 Streamlit App 💻 GitHub Repo 📄 Research Paper 📄 MS Excel Analysis

Project Overview

Analyzed customer churn behavior in European Bank across different segments.

Technologies Used

Python Pandas GroupBy Streamlit

Key Insights

  • The overall churn rate is 20%.
  • Customers with fewer products are most likely to leave the bank.
  • Customers aged between 31-45 are mostly likely to churn.
