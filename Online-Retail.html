🛍 Online Retail Customer Segmentation using Machine Learning & MLOps

End-to-End Machine Learning • MLOps • Customer Analytics • FastAPI • Streamlit

Project Overview

Developed an end-to-end Machine Learning Operations (MLOps) solution for customer segmentation using the Online Retail dataset. The project transforms transactional customer data into actionable business segments through RFM (Recency, Frequency, Monetary) analysis and K-Means clustering. The solution includes automated data preprocessing, model training, prediction logging, monitoring, drift detection, automated retraining, REST API deployment, and an interactive Streamlit dashboard for business users.

Streamlit App

Business Problem

Retail companies often struggle to identify valuable customers, detect declining customer engagement, and personalize marketing campaigns. Traditional segmentation methods are time-consuming and difficult to scale. This project automates customer segmentation, enabling data-driven marketing strategies while maintaining continuous model monitoring in production.

Project Objectives

  • Automate customer segmentation using machine learning.
  • Identify VIP, New, At-Risk, and Inactive customers.
  • Deploy a production-ready prediction API.
  • Build an interactive business dashboard.
  • Implement prediction monitoring and drift detection.
  • Enable automated model retraining when significant drift occurs.

Dataset Overview

The project uses the Online Retail transactional dataset consisting of customer purchase history, invoice information, product quantities, unit prices, and purchase dates. RFM features are engineered to capture customer purchasing behaviour.

Feature Engineering

  • Recency
  • Frequency
  • Monetary Value
  • Average Order Value
  • Customer Value

Machine Learning Solution

Customer segmentation was performed using the K-Means clustering algorithm. Customers were grouped into meaningful business segments based on purchasing behaviour and spending patterns.
