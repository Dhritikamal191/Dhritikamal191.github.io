End-to-End Machine Learning & MLOps

Intelligent House Price Prediction System

Developed a production-ready Machine Learning application that predicts residential property prices using advanced regression algorithms and an end-to-end MLOps pipeline. The solution integrates feature engineering, experiment tracking with MLflow, Docker containerization, CI/CD automation through GitHub Actions, real-time prediction logging with Supabase, and an interactive Streamlit dashboard for deployment and monitoring.

  • 🏡 House Price Prediction using Ames Housing Dataset
